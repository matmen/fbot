Settings


joinMessage (message)
The message that is sent when a user joins the server
Available placeholders: {MENTION}, {USERNAME}, {DISCRIM}, {TAG}, {SERVER}
leaveMessage (message)
The message that is sent when a user leaves the server
Available placeholders: {MENTION}, {USERNAME}, {DISCRIM}, {TAG}, {SERVER}
Note: Using mentions here is discouraged, because discord replaces left user's mentions with @invalid-user
messageChannel (#channel)
The channel that join and leave messages get sent to
Defaults to the server's initial #general channel
optOutOfAI (true/false)
Disables or enables the logging of specific messages for the AI command
Defaults to false (Not opt'ed out)
disableAutoreact (true/false)
Disables or enables the bot's auto reaction to specific messages
Defaults to false (Not disabled)
